Description\+:

state\+\_\+machine is a node used to communicate with all the other nodes. This one has a simple client, on /position\+\_\+server, in order to receive the next goal, a server, where receives a request by the user and an action client, which is used to send new goals and to check if a target is reached. Before start searching a new goal the state machine checks if there is a Command request from a client, then if the user command is \char`\"{}start\char`\"{}, the node make a request to the server receiving a goal between -\/5.\+0 and 5.\+0, regarding x and y position, and the theta orientation, then the state machine sends this goal to the server using the action client and waits for a result, which could be a \char`\"{}success\char`\"{} in case of goal reached and a \char`\"{}failure\char`\"{} in case of goal canceled. The node publishes this result on a specific topic.
